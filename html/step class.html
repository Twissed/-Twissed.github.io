skypiea.Step()[source]¶

Main Class for simulation parameters. Include beam, laser and plasma parameters at a given timestep

skypiea.Step.hist1D(self, xname, xconv=None, xrange=[None, None], bins=100, dx=None, plot=None, ax=None, yaxis='charge', **kwargs)¶

Plot/get 1D histogram data from the weighted beam.

Example

Simple plot of the x distribution (dQ/dx).¶
# Note that x axis is convert in um with xconv. step.sigma_x is the standard deviation (bunch length) in x.
H, xedges = step.hist1D('x',
                        xconv='um',
                        xrange=[-3*step.sigma_x*1e6,3*step.sigma_x*1e6],
                        bins=150,
                        plot='plot',
                        )
Multiple plots.¶
fig, axs = plt.subplots(2,3, figsize=(4*3,4*2), dpi=100, tight_layout=True)
fig.suptitle(f"dQ/dE with various options")

H, xedges = step.hist1D('energy_part',dx=1,plot='plot',ax=axs[0,0])
_ = axs[0,0].set_title(f"plot: dx = 1 ($\Delta$E = { xedges[1]-xedges[0] :.2f})")

H, xedges = step.hist1D('energy_part',bins=50,plot='plot',ax=axs[0,1])
_ = axs[0,1].set_title(f"plot: bins=50 ($\Delta$E = { xedges[1]-xedges[0] :.2f})")

H, xedges = step.hist1D('energy_part',bins=20,plot='hist',ax=axs[0,2])
_ = axs[0,2].set_title(f"hist: bins=20 ($\Delta$E = { xedges[1]-xedges[0] :.2f})")

H, xedges = step.hist1D('energy_part',bins=50,plot='lineplot',ax=axs[1,0])
_ = axs[1,0].set_title(f"lineplot: bins=50")

H, xedges = step.hist1D('energy_part',bins=40,plot='histplot',ax=axs[1,1])
_ = axs[1,1].set_title(f"histplot: bins=50")

H, xedges = step.hist1D('energy_part',dx=1,plot='histplot',ax=axs[1,2])
_ = axs[1,2].set_title(f"histplot: dx=1")
Parameters:

  • xname (str) – Name of the beam value for the x axis (e.g. “x”,”uy”,”energy_part”,…)

  • xconv (str, optional) – Unit wanted for the x axis. Defaults to None.

  • xrange (list, optional) – Range [min, max] of the data. Defaults to [None,None].

  • bins (int, optional) – Number of bins used. Defaults to 100.

  • dx (float, optional) – Step in x wanted. Modify bins. Defaults to None.

  • plot (str, optional) – Type of plot wanted. Defaults to None.

  • ax (matplotlib.pyplot.axes, optional) – matplotlib.pyplot.axes object for plots. Defaults to None. If None, but plot != None, will create a new figure with its own ax.

  • yaxis (str, optional) – Set the y axis. Defaults to ‘charge’.

  • **kwargs (List of properties) –

    • linestyle (str): Defaults to “-“.

Returns:

  • hist (float) (array) – The values of the histogram. See density and weights for a description of the possible semantics.

  • bin_edges (float) (array of dtype float) – Return the bin edges (length(hist)+1).
